ACT IV · KAIROS PLANNED

Execution, gated — by design.

Kairos is the planned execution layer. Strategies climb a ladder — backtest, paper, loopback, live — and every rung above backtest is a human gate. Today the ladder ends at backtest; the live rungs ship with Kairos, never before.

MCP TOOLS · FROM THE ACTUAL SERVER 
$ python -m digiquant.mcp_server

tools:
  digiquant_list_strategies        # registry inventory
  digiquant_run_backtest           # NautilusTrader run for one strategy
  digiquant_run_optimize           # parameter sweep / Bayesian search
  digiquant_run_pipeline           # backtest -> optimize -> export
  digiquant_export                 # tear sheets + artifacts
  digiquant_get_price_technicals   # indicator snapshot per ticker
  digiquant_get_macro_series       # FRED-backed macro series

Every capability is a discoverable MCP tool — point Claude, Cursor, or any MCP client at the server and ask for a backtest.
