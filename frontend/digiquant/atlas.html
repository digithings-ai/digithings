ACT I · CYCLES

Three temporal layers. One research library.

Atlas doesn't regenerate everything every day. Delta ops update only what changed. Full baselines run weekly. Monthly rollups capture the long arc. Each cycle writes structured JSON — not prose — to the shared document store.

DAILY · MON–SAT

Delta update

Triage phase decides what to regenerate vs. carry forward. Agents produce line-level edit ops — append, replace, or delete at specific document locations. Only the Digest is regenerated from scratch. Token cost: minimal.

delta-request.json daily_snapshots documents.digest triage phase
WEEKLY · SUNDAY

Full baseline

All 9 phases run in full. Research templates are regenerated from their YAML specs. Sets the week's anchor for downstream delta chains. More expensive, more thorough — run once per week.

9 phases run_type=baseline weekly anchor full regen
MONTHLY · ROLLUP

Long-term review

Cross-week pattern synthesis. Long-term macro framing, trend quality review, and post-mortem on evolution proposals. Produces a monthly digest document keyed to the ISO month.

monthly_digest pattern review evolution log lookback rollup
ACT II · PIPELINE

Nine phases. Parallel where it counts.

Baseline runs execute all phases in sequence — with Phase 1 running parallel sub-nodes across alt data sources. Each phase node returns a structured Pydantic output. No free-form text leaves the pipeline.

PHASE 1
Alt Data
Parallel ingestion: sentiment, flows, positioning, alternative signals. Each sub-node writes its segment output concurrently — LangGraph reducer merges without collision.
// parallel nodes
PHASE 2
Institutional
Institutional flow analysis — fund positioning, COT reports, dark pool prints, options flow. Structured verdict per segment.
PHASE 3
Macro
FRED series, Treasury yields, FX rates, crypto sentiment. Macro environment snapshot with structured bias outputs.
PHASE 4
Asset Class
Cross-asset view: equity, fixed income, commodities, crypto. Each class produces a structured bias with conviction rating.
PHASE 5
Equities — Sector Scorecard
Equity research flows into sector coverage, then a scored sector ranking. Parallel nodes per sector (XLK, XLF, XLE, XLV, XLI…). Scorecard output feeds Phase 7C analyst per-ticker work.
// equity → sectors → scorecard
PHASE 6
Consolidate
Cross-phase synthesis. Combines outputs from all parallel branches into a coherent macro picture. Resolves conflicts.
PHASE 7
Synthesis + Digest
Generates the canonical Digest — the primary daily artifact. Covers all sub-segments. Materialized to daily_snapshots.
PHASE 7C
Analyst — Per Ticker
One LLM call per watchlist ticker: bull/bear case, headwinds/tailwinds, conviction score. Fan-out width bounded by watchlist size.
// parallel per ticker
PHASE 7D
Portfolio Manager
PM-level briefing for Hermes: portfolio context, risk profile, per-asset verdicts from Phase 7C. Produces pm_allocation_memo.
PHASE 9
Evolution
Post-run quality review. Sources review, quality log, evolution proposals for next cycle. Runs on baseline and monthly only.

ACT III · DATA

Free data first. Always.

Atlas runs entirely on free-tier APIs and open data sources. No paid market data subscription required. Paid connectors are user-supplied.

FRED
Federal Reserve Economic Data
100+ macro series — yields, spreads, SOFR, inflation, labor. Free, 120 calls/min. Backfill to 1990.
TREASURY
US Treasury Yields
Official XML feed (Treasury.gov) + Yahoo Finance tickers ^IRX / ^FVX / ^TNX / ^TYX for reliable intraday fills.
PRICE TECH
Price & Technicals
OHLCV + computed technicals per watchlist ticker. Cached in Supabase price_history / price_technicals.
CRYPTO
CoinGecko + Fear & Greed
CoinGecko free tier (50 calls/min). Crypto Fear & Greed index for sentiment context. No API key required.
SEC / EDGAR
EdgarTools + MCP
XBRL parsing, 8-K / 10-Q / 10-K filings. No rate limits. MCP server available for DigiGraph agents and Claude Desktop.
FX
Yahoo Finance FX
Major currency pairs via Yahoo Finance. EUR/USD, GBP/USD, USD/JPY, USD/CHF and cross rates. Stored in macro_series_observations.

ACT IV · INTEGRATION

Persisted. Consumed downstream.

Every Atlas run writes structured JSON to Supabase. Hermes reads the Digest to run portfolio deliberation. Kairos taps theses to explore strategies. No intermediate files. No markdown pipelines.
