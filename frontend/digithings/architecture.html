// architecture

How the platform fits together.

A LangGraph supervisor at the center inspects every request and routes it to the right specialist — then writes the whole run to an immutable audit trail. Hover or focus a node to trace its connections; click through for the module reference.

// request lifecycle

Request in, audited run out.

01

Route

DigiGraph inspects the request and selects the specialist sub-graph — via a declarative tool registry, behind an OpenAI-compatible front door.

02

Execute

The specialist runs — quant research, vector retrieval, or chat — with LiteLLM routing/caching and LangGraph checkpoints that survive a restart.

03

Audit

Every span carries a correlation ID; PII is redacted before logs hit disk; the whole run lands in an immutable JSONL audit trail.
