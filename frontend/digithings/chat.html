// run it yourself
Three commands.

            
git clone https://github.com/digithings-ai/digithings
cd digithings
make up-digichat      # starts DigiGraph + DigiChat on port 3005
# visit http://localhost:3005

            
Requires Docker and an OPENAI_API_KEY or LiteLLM-compatible key in your environment. No account needed.
