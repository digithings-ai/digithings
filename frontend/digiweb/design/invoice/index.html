  5. 05 Economic & FX calendar

    Timezone‑correct release times, precise brief‑to‑calendar linking by event date, and consolidation onto a single‑source core economic calendar.

  6. 06 Custom 12X front-end

    Bespoke client-facing research dashboard: Today, Consensus, Matrix, Events and Trade‑Ideas tabs, with currency drill-downs, broker profiles and an events timeline.

  7. 07 Snapshots — Olympus data foundation

    Per‑run consensus and confluence snapshot tables that give the Olympus suite a queryable historical foundation.

  8. 08 CI / infrastructure & DevOps

    Per‑session cron workflows (Asia / London / New York), date‑backfill runs, an idempotent SQL migration runner, Supabase env standardisation, and CI permission fixes.

  9. 09 Documentation

    End‑to‑end architecture map, condensed Notion docs page, Olympus integration scope, and a documentation‑accuracy pass.
